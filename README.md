# Photo Gallery React Website

A modern React photo gallery website that displays a collection of images, allows users to view image details, read comments, share images, and explore related photos.

## Features

### Image Gallery Home Page

Displays all images from a central data source

Each image includes a short description

Clickable image and description navigate to a detailed view

### Single Image View Page

Full-size image display

Full description and title

Static comments section with “show more / show less”

Like, comment, and share interactions

Related images section

### Client-Side Routing

Navigation between pages using react-router-dom

Dynamic routes based on image ID

### Share Functionality

Toggle share options per image

Reusable ShareApps component
